About the parlour

A portfolio-ready concept for your web development review.

This project is designed to satisfy the course requirement of building a beautiful single-page website using frontend technologies while clearly demonstrating backend and database integration.

The interface highlights product discovery, brand storytelling, and customer enquiry handling in a structure that is easy for a mentor to review.

Frontend

HTML, CSS, and JavaScript deliver the single-page experience with animations, responsive sections, and dynamic rendering.

Backend

PHP endpoints handle menu fetching and form submission so the frontend does not talk to the database directly.

Database

MySQL tables store flavour items and customer enquiries for a complete workflow demonstration.
